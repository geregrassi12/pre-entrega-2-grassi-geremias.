Politica de privacidad

No recopilamos información personal!

Klee Games no recopila ninguna información que permita identificar individualmente a los usuarios. Klee Games puede recopilar información que los padres y educadores brindan voluntariamente a través del sitio web. Además, Klee Games puede recopilar información no personal sobre el uso de su sitio web y sus aplicaciones con el único fin de mejorar sus contenidos. Esta información se recopilará respetando las mejores prácticas para niños menores de 13 años. Cuando el usuario se ponga en contacto con nosotros para solicitar información o soporte, su dirección de correo electrónico se utilizará únicamente para proporcionar la información solicitada y no se almacenará. Cuando el usuario nos proporciona una dirección de correo electrónico para suscribirse a la lista de distribución de "Newsletters", el usuario confirma que esa dirección no es un dato personal, ni contiene ningún dato personal. Esta dirección sólo se utilizará para el envío de noticias de productos de Klee Games y nunca será vendida o cedida a terceros salvo en el caso de servicios de terceros estrictamente necesarios para el envío de dichos correos. Esta suscripción puede cancelarse en cualquier momento poniéndose en contacto con nosotros o utilizando el enlace habilitado para ello en los correos electrónicos recibidos. La lista de correo no es confidencial y no se utilizará para enviar información confidencial.
